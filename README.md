# 📔 Diário Online

Seu diário pessoal, simples e privado. Escreva sobre o seu dia, registre seu humor e guarde suas memórias — tudo fica salvo só no seu navegador, sem cadastro e sem servidor.

## O que faz

- **Escrever entradas** com data, título e texto livre
- **Humor do dia** — 6 estados, cada um com emoji e cor própria
- **Busca** por palavra no título ou no conteúdo
- **Editar** e excluir entradas
- Contador de entradas e de dias registrados
- **Salva sozinho** no navegador — suas memórias continuam lá
- 100% **responsivo** e **offline**

## Privacidade

Tudo o que você escreve fica **apenas no seu navegador** (`localStorage`). Nada é enviado pra nenhum servidor — é um diário de verdade, só seu.

## Stack

Next.js 16 (App Router) · TypeScript · Tailwind CSS v4 · Framer Motion · Lucide. Sem banco — as entradas ficam no `localStorage`.

## Rodar localmente

```bash
npm install
npm run dev
```

Abre em `http://localhost:3000`.
